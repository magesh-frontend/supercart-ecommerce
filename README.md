# 🛒 SuperCart — E-Commerce App
A simple e-commerce application built using React.js.
Users can browse products, view details, manage cart, and place orders with a clean UI.

---

Live Demo
https://cinesearch-movie-app-zo85.vercel.app/

## Features

* Browse all products instantly
* Product details page
* Add to cart / Remove from cart
* Update product quantity in cart
* Login & Signup (localStorage)
* Checkout with delivery form
* Order confirmation screen
* Responsive design

---

## Built With

* React.js
* React Router
* Context API
* CSS (inline styles)
* Fake Store API
* LocalStorage

---

## What I Practiced

In this project I worked on:

* React Hooks (`useState`, `useEffect`, `useContext`)
* API fetching with custom hook (`useFetch`)
* Context API for global state (Cart + Auth)
* React Router dynamic routes (`/product/:id`)
* Component based structure
* localStorage handling
